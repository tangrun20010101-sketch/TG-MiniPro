## 游戏界面设计（节点 3:8）

---

## 一、整体设计概述

游戏界面采用**中国传统春节主题**，色彩以红金为主，配合竹叶、灯笼等元素，营造喜庆氛围。整体风格类似《羊了个羊》的消除玩法，但具有鲜明的文化特色。

### 设计特点
- **主题**：春节、福牌、财神等吉祥元素
- **色彩**：红金渐变、青绿辅助（竹叶、边框）
- **布局**：顶部信息栏 + 中央牌堆 + 底部托盘 + 底部操作按钮

---

```

```

## 三、各区域详细设计

### 3.1 顶部栏 (Header)

| 元素 | 位置 | 设计说明 |
|------|------|----------|
| **菜单按钮** | 左侧 | 圆形红色按钮，金边，三条横线（汉堡菜单）图标 |
| **Score** | 中左 | 红色文字，显示当前分数（如 159032） |
| **Credit** | 中右 | 红色文字，显示代币/福签数量（如 23） |
| **设置按钮** | 右侧 | 圆形红色按钮，金边，齿轮图标 |

**样式规范：**
- 按钮：圆形，红色背景 (#FF4444 或相近)，金色描边 (#FFD700)
- 图标：白色或金色
- 文字：红色，醒目

---

### 3.2 背景

| 属性 | 说明 |
|------|------|
| **渐变** | 上：金橙色，下：深红橙色 |
| **装饰** | 竹干、抽象花纹、光斑效果 |
| **氛围** | 喜庆、魔法感、节日感 |

---

### 3.3 游戏主区域（福牌）

**牌堆特点：**
- 多层叠放，呈 3D 透视
- 仅顶层/未被遮挡的牌可点击
- 牌为圆角方形，有边框

**福牌类型（设计稿中出现）：**

| 类型 | 设计 | 边框 |
|------|------|------|
| 金龙/舞狮 | 红底金龙头 | 青蓝描边 + 米色框 |
| 福 | 绿色「福」字 | 青蓝描边 + 米色框 |
| 红灯笼 | 红灯笼图案 | 青蓝描边 + 米色框 |
| 元宝 | 金元宝，米色底 | 米色框 |
| 祥云 | 白色祥云，米色底 | 米色框 |
| 樱花 | 粉色樱花，米色底 | 米色框 |
| 财神 | 红袍财神持元宝，金 textures 底 | 特殊/稀有 |

**牌框样式：**
- 普通：米色底 + 青蓝描边
- 特殊：可能有红边等变化

---

### 3.4 托盘区域

| 属性 | 说明 |
|------|------|
| **外观** | 金色金属质感托盘，略凸起 |
| **装饰** | 左右两侧绿色竹叶 |
| **功能** | 放置选中的牌，最多 7 张 |
| **布局** | 牌横向排列，紧凑显示 |

---

### 3.5 底部操作按钮

| 按钮 | 图标 | 功能 |
|------|------|------|
| **移出** | 左向循环箭头 | 移除托盘中的牌 |
| **洗牌** | 两张重叠的牌 | 重新洗牌 |
| **撤回** | 左向弯曲箭头 | 撤销上一步 |

**样式：**
- 绿色圆形按钮 (#2E7D32 或相近)
- 金色描边
- 白色图标和文字

## 六、实现建议

### 6.1 布局
- 使用 `flex` 或 `grid` 实现顶部栏、主区域、托盘、底部按钮的垂直布局
- 主区域使用 `position: relative` 配合绝对定位放置牌

### 6.2 牌堆
- 3D 叠放可用 `transform: translateZ` 或多层 `z-index` + 偏移模拟
- 或使用 Canvas 绘制，便于控制层级和动画

### 6.3 托盘
- 使用 `tray_bg.png` 作为背景
- 牌槽用 `flex` 横向排列，间距均匀

### 6.4 背景
- 使用 `game_bg.png` 或 CSS 渐变 + 装饰图叠加
